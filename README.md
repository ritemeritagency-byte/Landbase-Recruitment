# Landbase Recruitment Website

Static multi-page website prepared for Vercel deployment.

## Local preview

Open `index.html` directly in your browser, or run a local static server.

## Deploy to Vercel

1. Push this folder to a GitHub repository.
2. In Vercel, click `Add New -> Project`.
3. Import your GitHub repository.
4. Keep framework preset as `Other`.
5. Build command: leave empty.
6. Output directory: leave empty (project root).
7. Click `Deploy`.

## Before going live

- Replace all placeholder WhatsApp numbers (`639171234567`) with your real number.
- Replace office address, email, and hours in `apply.html` with your final details.
